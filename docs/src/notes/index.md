* vue create -p dcloudio/uni-preset-vue 项目名称
* 然后将hbuilder里面的文件都拷贝到vue/cli项目下的src里面(除 unpackage、node_modules 目录)
* 在 vue-cli 工程内重新安装 npm 依赖（如果之前使用了 npm 依赖的话）
